## Overview

The CLI has two jobs:

| Guide | Use it for |
|------|-------------|
| [Registry](/docs/cli/registry) | `init`, `create`, `add`, `view`, `info`, and `docs` |
| [Backend](/docs/cli/backend) | `dev`, `codegen`, `env`, `deploy`, `aggregate`, `migrate`, `reset`, and `analyze` |

## Installation

```bash showLineNumbers
bun add kitcn
```

Backend resolves in this order:

1. `--backend <convex|concave>`
2. `backend` in `kitcn.json`
3. default `convex`
